# Colour System

The Lumina Design System is based on the official ETH Zurich Corporate Design colour palette.

For Lumina, the colour system is intentionally reduced.

The primary visual identity is built around:

- ETH Blue
- Black
- White

Secondary and accent colours may be used, but only with restraint.

ETH Blue should be the strongest colour signal of Lumina.

---

## Primary Colours

### ETH Blue

Primary Lumina colour.

Use ETH Blue for:

- key visual elements
- important UI cards
- semantic connections
- navigation elements
- selected highlights
- central Lumina components

HEX: `#215CAF`  
RGB: `33, 92, 175`

ETH Blue represents trust, precision, technology and institutional continuity.

It should be used more often than any other ETH corporate colour.

---

### Black

Use black for:

- typography
- icons
- high-contrast elements
- structural lines
- important labels

HEX: `#000000`  
RGB: `0, 0, 0`

Black provides clarity and contrast.

---

### White

Use white for:

- primary backgrounds
- whitespace
- cards
- clean presentation areas
- calm web layouts

HEX: `#FFFFFF`  
RGB: `255, 255, 255`

White is not empty space.

White is an active part of the Lumina visual identity.

---

## Secondary Colours

### ETH Petrol

Use ETH Petrol for:

- data flows
- APIs
- metadata structures
- semantic relationships
- technical but calm visual accents
- secondary interface elements

HEX: `#007894`  
RGB: `0, 120, 148`

ETH Petrol is the preferred secondary colour for technical and data-oriented visuals.

---

### ETH Grey

Use ETH Grey for:

- secondary UI elements
- background structures
- labels
- supporting diagrams
- neutral information layers

HEX: `#6F6F6F`  
RGB: `111, 111, 111`

ETH Grey helps create hierarchy without adding unnecessary colour.

---

## Accent Colours

Accent colours should be used sparingly and only when they add semantic meaning.

### ETH Green

Use ETH Green for:

- successful enrichment
- validated metadata
- completed workflows
- positive outcomes
- quality improvements

HEX: `#627313`  
RGB: `98, 115, 19`

---

### ETH Purple

Use ETH Purple for:

- advanced AI capabilities
- semantic intelligence
- experimental features
- innovation themes

HEX: `#A7117A`  
RGB: `167, 17, 122`

Use ETH Purple carefully. It should not dominate Lumina visuals.

---

### ETH Bronze

Use ETH Bronze for:

- categorisation
- special highlights
- selected knowledge domains
- subtle emphasis

HEX: `#8E6713`  
RGB: `142, 103, 19`

---

### ETH Red

Use ETH Red only for:

- warnings
- errors
- critical states
- exceptional situations

HEX: `#B7352D`  
RGB: `183, 53, 45`

ETH Red is part of the ETH corporate palette, but it is not part of the normal Lumina visual identity.

Do not use ETH Red as a general accent colour.

## Colour Principles

Use colours with restraint.

Prefer compositions dominated by white, black and ETH Blue.

Normally combine no more than two ETH corporate colours within one illustration, excluding black, white and grey.

Use ETH Petrol as the default secondary colour for technical and data-related elements.

Use accent colours only when they support meaning.

Avoid rainbow-like combinations.

Avoid decorative colour use without purpose.

Avoid colourful SaaS-style gradients that weaken the ETH identity.

Whitespace is essential.

A Lumina visual should feel calm before it feels colourful.

---

## Accessibility Notes

Use strong contrast for text and important interface elements.

Use white text on dark or saturated colour backgrounds.

Use black text on very light colour shades.

Avoid placing small text on medium colour shades.

If a generated visual contains text, review contrast manually before using it in presentations or web contexts.
